# Technology Stack 📲📲📲

### Front-End:

- **NextJs (v13+)** on **React v.18**
- Tailwind css.
- Framer motion.
- SignalR Node-client.
- Auth0 m2m

### Back-End:

- **Web API Dotnet** on **.Net 8.0**.
- **SignalR**
- Entity Framework core.
- Authentication with Auth0.
- Serilog

### Data-Layer:

- Mssql:v2019

### Dockerized
